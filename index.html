Q1. How does javascript work?

Ans: Javascript is a client side language. It runs inside a browser via JS engines. JavaScript access the HTML and CSS file via DOM. The browser loads HTML and CSS file first to show the page. Then the JS engine loads JavaScript code as order as it written.



Q2. Difference Between Local Storage, Session Storage

Ans: Local Storage and Session Storage are almost same thing. But there is only one main Difference between this two thing. That is, local storage keeps the data for long time. One the other hand, session storage keeps the data for that session only. If user close that tab or browser session storage will erase the data.
